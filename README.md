# Python File Organizer for Windows

A simple Python script that automatically organizes files in a folder into categories based on file type.

## Features
- Organizes files by extension
- Creates folders automatically
- Supports images, documents, videos, music, and archives
- Easy to customize

## Requirements
- Python 3.x

## How to Run

1. Place the script in the folder you want to organize
2. Run:

```bash
python organizer.py
```
